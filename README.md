# FocusCLI

FocusCLI is a command-line tool designed to help you manage time and improve focus using techniques like the Pomodoro Technique. It allows you to track work sessions and breaks, providing notifications and storing session data for review.

## Features

- Start a focus session of specified duration.
- Take a break of specified duration.
- Receive desktop notifications when sessions and breaks end.
- Persist session data for tracking.
